# 📖 AirTracer 用户使用指南

欢迎使用 **AirTracer** 轨迹追踪与设备管理平台！本文档将指导您从零开始，完成账号注册、设备接入、轨迹监控以及报警通知等核心功能的配置。

---

## 1. 账号注册与登录

1. 打开浏览器，访问官方在线平台：[https://airtracer.us](https://airtracer.us)。
2. 点击首页的 **注册 (Register)**，填写您的邮箱和密码，完成账户创建。
3. 注册成功后返回登录界面，输入凭据进入 **控制中心 (Dashboard)**。

> [!NOTE]
> 控制中心是整个系统的核心看板，您可以在此直观地查看所有设备的最新位置、电量状态，并在地图上实时追踪它们。

---

## 2. 添加与管理设备

在左侧导航栏中选择 **设备管理 (Device Management)**，点击 **添加设备 (Add Device)**。AirTracer 支持接入多种不同协议的设备：

### 📱 方案 A：接入智能手机作为追踪器 (GPS 设备)
最简单的测试定位方法是利用您手头上的智能手机：
1. 在手机上安装标准的定位上报客户端（如 Traccar Client，可在各大应用市场下载）。
2. 在 AirTracer 的“添加设备”弹窗中，选择设备类型为 **GPS 设备 (Traccar)**。
3. 系统会为您生成一个唯一的 **设备识别号 (Unique Identifier)**。
4. 打开手机上的 Traccar Client：
   * **服务器地址**：填入 `http://api.airtracer.us:5055`（具体以页面提示为准）。
   * **设备识别号**：填入 AirTracer 生成的识别号。
   * **定位精度与频率**：建议设置为每 30 秒上报一次。
5. 开启手机客户端的“定位服务”开关，您的手机即可源源不断地上报 GPS 坐标到平台。

### 📡 方案 B：接入 FindMy 蓝牙定位标签
如果您拥有兼容 Apple FindMy 网络的蓝牙定位标签（如 InfiTag 或 LoopTag），可通过该网络实现全球免流定位：

1. **选择设备类型**：在添加设备时，选择设备类型为 **FindMy**。
2. **选择密钥模式 (InfiTag vs LoopTag)**：
   * **InfiTag (无极滚动密钥) 【推荐】**：抛弃了固定公钥模式，采用**动态滚动密钥算法**。其定位更实时、轨迹点更多更密集、路线渲染更详细。同时避免了固定 Key 因被 Apple 判定为垃圾数据而丢弃位置的问题，定位精度最高。接入时需要输入您的设备 **Seed (种子十六进制字符)**。
   * **LoopTag (传统固定 Key)**：基于传统的固定公钥模式。适合简易测试或低频定位。接入时输入固件生成的 **初始公钥 (Public Key)**。注：固定 Key 模式有可能会被苹果公司过滤或丢弃一些位置报告，导致轨迹有断连。
3. **完成绑定**：保存后，后端系统将根据您的设置，定期检索设备在 FindMy 网络中的最新位置。

## 3. 轨迹分析与回放

1. **实时追踪**：在“控制中心”的地图上，点击特定设备，可以查看该设备的电量、最后在线时间、速度以及当前经纬度。地图会随着设备位置更新自动平移。
2. **历史轨迹**：
   * 进入 **轨迹分析 (Track History)** 页面。
   * 选择目标设备及要查询的时间段（如“今天”、“昨天”或“自定义范围”）。
   * 点击 **查询**，地图上将呈现出一条连贯的定位移动轨迹线。
   * 您可以点击 **播放 (Play)** 按钮观看轨迹的动态回放，调整播放速度。
   * 系统会自动识别**停留点 (Dwell Points)**，在地图上以图标形式展现并标明停留时长。

---

## 4. 电子围栏 (Geofencing)

电子围栏功能允许您在地图上划定安全区域，并在设备进出该区域时触发报警。

1. 进入 **电子围栏 (Geofences)** 页面。
2. 点击 **新建围栏 (Create Geofence)**。
3. 在地图上点击并拖动，绘制一个圆形或多边形区域，输入围栏名称（例如：“家”、“公司”）。
4. 选择该围栏适用的设备。
5. 选择触发条件：**进入时报警**、**离开时报警** 或 **两者皆报警**。
6. 点击 **保存**。

---

## 5. 通知配置 (Telegram 机器人集成)

为了在设备进出围栏或电量过低时获得即时推送，您可以配置 Telegram 机器人。

### 第一步：创建您自己的 Telegram Bot
1. 在 Telegram 中搜索 `@BotFather` 并启动对话。
2. 发送命令 `/newbot`，按照提示为机器人命名，您将获得一个 **HTTP API Token**（格式如 `123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ`）。
3. 搜索并启动您的机器人，向它发送任意一条消息。
4. 搜索并关注 `@userinfobot`，发送任意消息获取您的 **User ID** (Chat ID，一串纯数字)。

### 第二步：在 AirTracer 平台配置
1. 登录 AirTracer，在导航栏中点击 **系统设置 (System Settings)**（或 **个人中心 -> 通知配置**）。
2. 找到 **Telegram 报警配置** 板块。
3. 填入您的 **Bot Token** 和 **Chat ID**。
4. 点击 **保存并测试连接 (Test Connection)**。
5. 如果配置正确，您的 Telegram 此时会收到一条来自机器人的测试成功消息。
6. 此后，任何电子围栏触发事件或设备低电量警告，都会通过该 Telegram 机器人即时推送到您的手机上。
