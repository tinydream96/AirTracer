# 高德地图 API Key 申请教程

本教程将手把手教你如何免费申请一个高德地图的 API Key（密钥）以及安全密钥（Security Code），以便在项目中使用地图和定位相关服务。这样你就可以看到定位的具体地址，比如某某街道，某某小区等。整个过程非常简单，大约只需要 3-5 分钟。

## 第一步：注册并登录高德开放平台

1. 必须使用电脑打开浏览器，访问 **[高德开放平台官网](https://lbs.amap.com/)**。
2. 点击页面右上角的 **“登录”** 或 **“注册”**。
   - 💡 **提示**：推荐直接使用**支付宝扫码快捷登录**，这样后续实名认证也会更方便。你也可以选择用手机号进行注册。

![高德地图官网](../images/amap%20api%20key/高德API%20key-1.png)
![登录注册](../images/amap%20api%20key/高德API%20key-2.png)

## 第二步：完成开发者实名认证

为了正常调用高德的地图接口，你需要完成开发者实名认证。

1. 登录成功后，系统通常会提示你进行认证。如果没有提示，可以点击页面右上角的个人头像，选择 **“开发者认证”**。

![开发者认证](../images/amap%20api%20key/高德API%20key-3.png)

1. 认证类型请选择 **“个人开发者”**（如果是为公司项目申请，则选择企业开发者，但这通常需要营业执照）。
2. 按照页面提示填写你的姓名、身份证号等基本信息，并使用支付宝进行扫码人脸识别认证。
3. 认证一般是秒过的，通过后你就可以正式使用高德的服务了。

## 第三步：进入控制台

1. 在高德开放平台首页，点击右上角的 **“控制台”** 按钮。
   - （控制台就是你的后台管理系统，以后管理 Key 和查看调用量都在这里）。

![进入控制台](../images/amap%20api%20key/高德API%20key-4.png)

## 第四步：创建新应用

要想获取 Key，首先得建一个“应用”来存放这个 Key。

1. 在控制台左侧的菜单栏中，依次点击 **“应用管理”** -> **“我的应用”**。
2. 在页面右上角点击蓝色的 **“创建新应用”** 按钮。

![创建应用](../images/amap%20api%20key/高德API%20key-5.png)
3. 在弹出的窗口中填写应用的基本信息：

- **应用名称**：自己随便起一个容易记住的名字，比如“我的测试项目”、“轨迹追踪App”等。
- **应用类型**：根据你的实际情况随便选一个最接近的类别，比如“出行”、“工具”或“其他”。

1. 填完后，点击下方的 **“创建”**。

![填写应用信息](../images/amap%20api%20key/高德API%20key-6.png)

## 第五步：为应用添加 Key

应用创建好之后，就可以在它下面生成具体的 Key 了。

1. 在“我的应用”列表中，找到你刚才创建的应用。
2. 点击该应用右上角的 **“添加”** 按钮。

![添加 Key](../images/amap%20api%20key/高德API%20key-7.png)
3. 在弹出的页面中填写 Key 的配置信息：

- **Key 名称**：随便填，比如“Web端测试Key”、“后端调用Key”等。
- **服务平台** 请选择 **“Web服务”**。

1. 填完后，点击下方的 **“提交”**。

![填写 Key 配置](../images/amap%20api%20key/高德API%20key-8.png)

## 第六步：获取并使用你的 API Key

1. 提交成功后，页面会自动跳转回“我的应用”列表。
2. 展开你的应用，你会看到刚刚添加的 Key 已经生成了。
3. 里面包含重要的信息：
   - **Key**：一串长长的字母和数字组合（例如：`3a5b6c7d...`）。

![获取 Key](../images/amap%20api%20key/高德API%20key-9.png)

1. 将Key填入我们的系统设置中！

![复制 Key](../images/amap%20api%20key/高德API%20key-10.png)

---

### ⚠️ 重要注意事项

1. **妥善保管**：千万不要把你的 Key 和安全密钥发给陌生人，也不要直接暴露在开源的 Github 代码仓库中，以免被别人盗刷你的额度。
2. **免费额度**：高德对个人开发者每天都提供了一定数量的免费调用额度，对于日常学习测试和小型个人项目来说完全足够了，不用担心收费问题。
